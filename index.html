Prática Integradora Desenvolvimento de Apps

  1. Criar repositório no github
  2. Adicionar um index.html com um conteúdo básico de html
  3. Conectar o repositório local com o github
  4. Adicionar alteração
  5. Fazer commit
  6. Criar pasta images
  7. Adicionar imagem no index.html
  8. Adicionar alteração com git add
  9. Fazer um novo commit com git commit -m
